## Overview

This endpoint returns a JSON summary of sales data for dashboard visualizations and analytics. It provides:
- Monthly sales totals for the last 6 months
- Top 5 best-selling products
- Overall sales statistics (total revenue, order count, average ticket)

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication. User must have one of the following roles:
  - **Administrador**
  - **Gerente**
  - **Supervisor**
</ParamField>

<ResponseField name="status" type="string">
  Response status indicator
</ResponseField>

<ResponseField name="data" type="object">
  Container for all sales data
  
  <Expandable title="properties">
    <ResponseField name="resumen" type="object">
      General sales statistics
      
      <Expandable title="properties">
        <ResponseField name="total_ingresos" type="number">
          Total revenue from all sales (calculated as sum of `cantidad * precio_unitario` from order details)
        </ResponseField>
        
        <ResponseField name="total_pedidos" type="number">
          Total count of orders
        </ResponseField>
        
        <ResponseField name="ticket_promedio" type="number">
          Average order value (total_ingresos / total_pedidos)
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="grafica_ventas" type="array">
      Monthly sales data for the last 6 months
      
      <Expandable title="array items">
        <ResponseField name="mes" type="number">
          Month number (1-12)
        </ResponseField>
        
        <ResponseField name="total" type="number">
          Total sales for that month
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="top_productos" type="array">
      Top 5 best-selling product variants
      
      <Expandable title="array items">
        <ResponseField name="variante_id" type="number">
          ID of the product variant
        </ResponseField>
        
        <ResponseField name="total_vendido" type="number">
          Total quantity sold
        </ResponseField>
        
        <ResponseField name="variante" type="object">
          Variant details including nested product information
          
          <Expandable title="properties">
            <ResponseField name="id" type="number">
              Variant ID
            </ResponseField>
            
            <ResponseField name="producto" type="object">
              Parent product information
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

```bash
curl -X GET http://127.0.0.1:8000/api/reportes/ventas \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Accept: application/json"
```

## Error Responses

### 401 Unauthorized

```json
{
  "message": "Unauthenticated."
}
```

### 403 Forbidden

```json
{
  "message": "No tiene permisos para acceder a este recurso."
}
```

Returned when the authenticated user does not have Administrador, Gerente, or Supervisor role.

## Implementation Notes

- Monthly sales are calculated from `DetallePedido` records using `MONTH(creado_en)`
- Top products are determined by sum of `cantidad` grouped by `variante_id`
- Total revenue calculation: `SUM(cantidad * precio_unitario)`
- The endpoint uses eager loading for product variant relationships
- Results are ordered by most recent month first for `grafica_ventas`
- Controller location: `app/Http/Controllers/Api/ReporteController.php:17`
