## Endpoint

```
GET /api/categorias
```

## Description

Retrieves a list of all product categories available in the system.

## Authentication

<Note>
This is a public endpoint. No authentication required.
</Note>

## Response

<ResponseField name="status" type="string">
  The status of the response. Returns `"success"` on successful request.
</ResponseField>

<ResponseField name="data" type="array">
  Array of category objects.
  
  <Expandable title="categoria object">
    <ResponseField name="id" type="integer">
      Unique identifier for the category.
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      The name of the category.
    </ResponseField>
    
    <ResponseField name="descripcion" type="string">
      Description of the category.
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Timestamp when the category was created.
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Timestamp when the category was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.solare.com/api/categorias \
  -H "Accept: application/json"
```
