## Introduction

SOLARE API is a comprehensive Laravel 12-based RESTful API designed for managing a luxury furniture e-commerce platform. Built with enterprise-grade security and scalability in mind, it provides a complete backend solution for product catalogs, order management, inventory control, and sales reporting.

## Architecture Overview

SOLARE API follows a clean, layered architecture built on Laravel best practices:

- **RESTful Design**: JSON-based API with standard HTTP methods
- **Database Layer**: MariaDB with custom timestamp columns and triggers
- **Authentication**: Sanctum/Passport with token-based authentication
- **Authorization**: Custom middleware for role-based access control
- **Business Logic**: Service-oriented controllers with transaction support
